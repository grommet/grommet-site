To run this application, execute the following commands:

  1. Install NPM modules

  ```
  $ npm install (or yarn install)
  ```

  2. Start the front-end dev server:

  ```
  $ npm run start
  ```

  3. Create the app distribution

  ```
  $ npm run build
  ```

  4. Test and run linters:

  ```
  $ npm test
  ```
